# 🛍️ Black Friday Sales Analysis

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the Black Friday Sales dataset to understand customer purchasing behavior during the Black Friday shopping event.

The analysis focuses on identifying patterns related to **gender, age group, marital status, occupation, and city category**, and how these factors influence purchase amounts.

The insights derived from this analysis can help businesses improve **targeted marketing strategies, product recommendations, and customer segmentation**.

---

## 📂 Dataset Information

The dataset contains transactional data of customers who purchased products during the Black Friday sale.

### Dataset Features

| Column                     | Description                                     |
| -------------------------- | ----------------------------------------------- |
| User_ID                    | Unique ID assigned to each customer             |
| Product_ID                 | Unique ID assigned to each product              |
| Gender                     | Gender of the customer                          |
| Age                        | Age group of the customer                       |
| Occupation                 | Occupation code of the customer                 |
| City_Category              | Category of the city (A, B, C)                  |
| Stay_In_Current_City_Years | Number of years customer has stayed in the city |
| Marital_Status             | Marital status of the customer                  |
| Product_Category_1         | Primary product category                        |
| Product_Category_2         | Secondary product category                      |
| Product_Category_3         | Third product category                          |
| Purchase                   | Purchase amount                                 |

Dataset Source:** Black Friday Sale Dataset**

## 📊 Analysis Performed

The following relationships were analyzed in the dataset:

* Gender vs Purchase Behavior
* Age Group vs Purchase Amount
* Marital Status vs Spending Pattern
* Occupation vs Average Purchase
* City Category vs Total Sales
* Product Category Demand

Visualizations such as **bar plots, count plots, and correlation heatmaps** were used to better understand the data.

---

## 📈 Key Insights

* Male customers contributed significantly more to total purchases.
* Customers aged **26–35 years** made the highest number of purchases.
* Some occupations showed **higher average spending per transaction**.
* **City Category B** generated a large portion of the sales.
* Certain product categories were consistently purchased more than others.

These insights can help businesses design **better marketing campaigns and product strategies** during major sale events.
